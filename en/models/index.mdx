## Naming convention

Todos los modelos siguen el formato `<provider>/<model-slug>`:

- `anthropic/claude-sonnet-4-6`
- `openai/gpt-5`
- `google/gemini-2.5-pro`
- `fal/flux-1.1-pro`
- `xai/grok-4`

El namespace deja claro qué proveedor está detrás y te permite cambiar entre alternativas con solo cambiar 1 string.

## API live de modelos

Para obtener la lista actualizada al momento, llama:

```bash
curl https://api.geekhub.mx/v1/models \
  -H "Authorization: Bearer ghub_sk_live_xxx"
```

Te devuelve algo como:

```json
{
  "object": "list",
  "data": [
    {
      "id": "anthropic/claude-sonnet-4-6",
      "object": "model",
      "kind": "chat",
      "owned_by": "anthropic",
      "context_length": 200000,
      "pricing": {
        "input_per_1m_usd": 3,
        "output_per_1m_usd": 15
      }
    },
    ...
  ]
}
```

## Pricing

Los precios listados son **USD del proveedor original**. Para el costo final:

```
costo_final_mxn = precio_provider × (1 + 0.05) × FX_MXN_USD
```

- `+5%` markup de Geek Hub
- `FX_MXN_USD` se fija mensualmente y se anuncia a clientes (sin sorpresas)
