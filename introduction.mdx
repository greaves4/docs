## Estructura del API

Todos los endpoints son OpenAI-compatible donde aplica:

```
POST /v1/chat/completions       Chat con cualquier modelo
POST /v1/images/generations     Generación de imágenes
POST /v1/videos/generations     Generación de video (async)
GET  /v1/videos/{id}            Estado de un job de video
POST /v1/audio/speech           Text-to-Speech (streaming)
GET  /v1/models                 Catálogo completo
GET  /health                    Health check (público)
```

## Base URL

```
https://api.geekhub.mx
```
