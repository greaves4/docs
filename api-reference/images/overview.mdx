El endpoint `POST /v1/images/generations` es compatible con la API de OpenAI Images. Cambias el `model` y eliges de nuestro catálogo.

## Storage propio

Cada imagen generada se sube **automáticamente** a tu Supabase Storage (`supabase.geekhub.mx`) en un bucket público con URLs por UUID — efectivamente impossible de adivinar (mismo modelo que Cloudinary o fal.media).

Esto significa que las URLs **no expiran** y puedes referenciar la imagen desde:

- Tu app web (`<img src={url} />`)
- Slack, email, redes sociales
- Backup propio descargando los bytes

Si quieres almacenarlas en TU storage en lugar del nuestro, descárgalas inmediatamente con cualquier HTTP client:

```bash
curl -O https://supabase.geekhub.mx/storage/v1/object/public/generated-images/.../0.png
```
